CORES FEATURES

Collaborative Requirement Engineering System has the following Features:
  • MAIN
  • MONITORING

Projects

Registered member can create Project which can be private or Public Project, then mambers can contribute on that project.

Viewpoint

Project Member can Create Viewpoint based on a particular Project then other members can contribute resources and comments on that viewpoint.

Goals

Member of the Project can create Goals based on particular Viewpoint, where by other members can contribute resources and comments on that Goal, also Goals are category based.

Requirement

Project Members can create, view, and contribute resources and comments on Requirement based on particular Goal that Requirement belongs to.

Scenario

Project Member can view, create and contribute resources and ideas on Scenario based on a perticular Requirement.

Process and Usecase

Project members can create view, and contribute on Process also in Usecase of the project.

Invite Members

If a project is Private Project Admin (Creator) can Invite members to Join the Project for contributions.

Membership Requests

If the Project is Private and Discoverable, system users can ask to join a Project, Project admin can accept or Reject a request.

Project Members

Project admin can view Project Members details, Suspend,even remove them from the project when necessary.

Project Incentives

Project Creator can provide Incentives to Contributors of his/her project. this gift can be monetary gift or other things.
